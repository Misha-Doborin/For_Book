# Интерфейс для оживления картинки

Небольшое desktop-приложение на Tkinter для сценария: пользователь загружает исходный файл, вводит профессию, создаёт видео и скачивает результат.

Сюжет промпта закреплён в интерфейсе: маленький мальчик высоко подпрыгивает, а после приземления оказывается в выбранной профессии.

## Запуск

```bash
python3 main.py
```

## Что уже есть

- выбор исходной картинки или видео;
- поле ввода профессии;
- автоматическое формирование промпта для генерации;
- кнопка создания результата;
- кнопка скачивания видео.

## Важно про генерацию

Сейчас в проекте реализован интерфейс и демо-рендер без внешних зависимостей. Если пользователь загружает видео, приложение копирует его как результат и сохраняет рядом текстовый промпт для будущего backend. Для настоящего оживления картинки нужно заменить метод `RenderService.render` в `main.py` на вызов модели или API генерации видео.
