# Интерфейс для оживления картинки

Desktop-приложение на Tkinter для схемы: пользователь загружает **картинку**, вводит профессию, создаёт видео и скачивает готовый MP4.

Сюжет промпта закреплён в интерфейсе: маленький мальчик с загруженной картинки высоко подпрыгивает, а после приземления оказывается в выбранной профессии.

## Запуск

```bash
python3 main.py
```

## Что уже есть

- выбор исходной картинки: PNG, JPG, WEBP, BMP или GIF;
- поле ввода профессии;
- автоматическое формирование промпта для image-to-video генерации;
- кнопка создания видео;
- кнопка скачивания MP4.

## Важно про генерацию

Интерфейс уже построен именно под поток **картинка → видео**. Текущий демо-рендер без внешнего AI backend создаёт короткий MP4 из загруженной картинки через `ffmpeg`, если он установлен в системе, и сохраняет рядом текстовый промпт. Для настоящего превращения мальчика в выбранную профессию нужно заменить метод `RenderService.render` в `main.py` на вызов модели или API image-to-video генерации.
